Introduccion HTML

¿Que es HTML?

  • HTML son las siglas de Hyper Text Markup Lenguage (Lenguaje De Marcado De Hypertexto)
  • HTML es el estandar de lenguaje de marcado para crear paginas web
  • HTML describe la estructura de una pagina web
  • HTML consiste en una serie de elementos
  • HTML le dice al navegador como mostrar el contenido
  • HTML etiqueta los contenidos diciendo "Esto es un encabezado", "Este es un link", "Esto es un parrafo" etc.

¿Que es un Elemento HTML?

Un Elemento HTML es definido por una etiqueta de inicio, algun contenido, y una etiqueta de cierre

<etiqueta> Contenido </etiqueta>
Etiqueta inicial Contenido Etiqueta de cierre
<h1> Mi primer encabezado </h1>
<p> Mi primer parrafo </p>
