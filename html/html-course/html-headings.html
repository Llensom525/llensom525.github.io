Titulos HTML

Los encabezados en HTML son titulos o subtitulos que puedes mostrar en una pagina web

Ejemplo

Titulo 1

Titulo 2

Titulo 3

Titulo 4

Titulo 5
Titulo 6

Los titulos son definidos de la etiqueta <h1> hasta la <h6>

Nota: Los navegadores añaden automaticamente un margin despues de cada encabezado

Los encabezados son importantes

Los motores de busqueda usan los titulos para indexar la estructura y contenido de una pagina web

Ademas de que los usuarios a menuda navegan por las paginas basandose en sus titulos. Es importante utilizar los titulos para mostrar la estructura del documento

<h1> son usados para los titulos principales, seguido de los titulos <h2>, luego el menos importante <h3> y asi sucesivamente

Nota: Usar los titulos solo para titulos, no para hacer un texto grande o en negrita

Titulos grandes

Cada titulo HTML tiene un tamaño por defecto. Como sea tu puedes especificar el tamaño de cualquier titulo con wl atributo style usando la propiedad CSS font-size
