Formato de texto HTML

HTML contiene muchos elementos para definir el texto con un significado especial

Elementos de formato HTML

Los elementos de formato fueron diseñados para mostrar un tipo especial de texto

  • <b> - Bold text
  • <strong> - Important text
  • <i> - Italic text
  • <em> - Emphasized text
  • <mark> - Marked text
  • <small> - Smaller text
  • <del> - Deleted text
  • <ins> - Inserted text
  • <sub> - Subscript text
  • <sup> - Superscript text

Elementos <b> y <strong>

El elemento <b> define un texto en negrita, sin ninguna importancia extra

Ejemplo

<b>Este texto esta en negrita</b>
Este texto esta en negrita

El elemento <strong> define un texto con gran importancia. El contenido dentro de este es tipicamente mostrado en negrita

Ejemplo

<strong>Este texto es importante!</strong>
Este texto es importante

Elementos <i> y <em>

El elemento <i> define una parte el texto en una voz o estado alternativo. El contenido dentro de el tipicamente es italic (curiva)

Tip: La etiqueta <i> Normalmente se utiliza a menudo para indicar un termino tecnica, una frase en otro idioma,un pensamiento, el nombre de un barco, etc.

Ejemplo

<i>Soy un pensamiento, hmmmm</i>
Soy un pensamiento, hmmmm

El elemento <em> define un texto enfatizado. EL contenido en su interior normalmente se muesta en italic (cursiva)

Tip: Los lectores de pantalla pronuncian las palabras en <em> con una enfasis, usando estres verbal

Elemento <small>

El elemento <small> define un texto pequeño

Ejemplo

<small>Soy un texto pequeño</small>
Soy un texto pequeño

Elemento <mark>

El elemento <mark> define un texto que se debe marcar o resaltar

Ejemplo

<p>No olvides comprar la <mark>leche</mark> en el supermercado</p>

No olvides comprar la leche en el supermercado

Elemento <del>

Define un texto que ha sido borrado del documento

Ejemplo

<p>Mi color favorito es el <del>rojo</del></p>

Mi color favorito es el rojo

Elemento <ins>

Define un texto que se ha insertado en un documento

<p>Mi color favorito es el <del>rojo</del> <ins>blanco</ins></p>

Mi color favorito es el rojo blanco

Elemento <sub>

Este elemento define el texto de subindice, aparece medio caracter debajo de lo normal y a veces se representa en una fuente mas pequeña. Se puede utilizar para formulas quimicas como el H2O

Ejemplo

<p>Este es un texto de <sub>subindice</sub></p>

Este es un texto de subindice

Elemento <sup>

El elemento <sup> aparece medio caracter por encima de la linea normal y a veces se representa con una fuente mas pequeña. Se puede utilizar para las notas de pie como WWW[1]

Ejemplo

<p>Este es un texto de <sup<superindice</sup></p>

Este es un texto de superindice
