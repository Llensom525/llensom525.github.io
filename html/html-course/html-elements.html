Elementos HTML

El elemento HTML es todo, desde la etiqueta inicial hasta la de cierre

<etiqueta>Contenido</etiqueta>

Nota: Algunos elementos no tienen contenido (Como el elemento <br>). Estos se llaman elementos vacios. Los elementos vacios no cuentan con etiqueta de cierre

Elementos HTML anidados

Los elementos HTML pueden ser anidados, esto significa que pueden contener otros elementos

Todos los documentos HTML consisten en elementos anidados

El siguiente ejemplo contiene 4 elementos HTML: <html>, <body>, <h1>, <p>

Nunca olvidar la etiqueta de cierre

Algunos elementos se mostraran correctamente si olvidas la etiqueta de cierre, pero de todas formas nunca la olvides, ya que se podrian ocasionar errores inesperados

Elementos HTML vacios

Elementos HTML sin contenido son llamados elementos vacios.

La etiqueta <br> define un salto de linea, y es un elemento vacio sin etiqueta de cierre

HTML no es sensible a las mayusculas

En HTML puedes poner <P> y <p>, ya que los dos para HTML es lo mismo
