Links HTML

Los links estan en casi todas las paginas web. Estos permiten al usuerio hacer click y llevarlos de una pagina a otra

Hyperlinks

Los links HTML son hipervinculos

Tu puedes hacer click a un link e ir a otro documento

Cuando tu mueves tu mouse sobre un link, la flecha del mouse se convertira en una pequeña mano

Nota: Un enlace no tiene porque se un texto, puede ser una imagen o cualquier otro elemento HTML

Sintaxis

La etiqueta <a> define un hipervinculo. Tiene la siguiente sintaxis

<a href="link text"></a>

Lo más importante del atributo <a> es el elemento href que indica el link de destino

El texto del enlace es la parte que sera visible para el lector

Al clickear en el texto, el lector sera enviado a la direccióon URL especificada

Ejemplo

<a href="https://www.llensomlle.com">Visita Llensom Lle!</a>

Nota: Por defecto los links apareceran de la siguiente forma en todos los navegadores:

  • Un link no visitado, subrayado y azul
  • Un link visitado, subrayado y purpura
  • Un link activo, subrayado y rojo

Pero puedes darles tu propio formato con CSS

El atributo target

Por defecto, la pagina vinculada sera mostrada in la pagina actual del navegador. Para cambiar esto debemos especificar otro objetivo para el link

El atributo target especifica donde abrira el documento vinculado

Este atributo tiene los siguientes valores

  • _selfDefault. Abre el documento en la misma ventana/pestaña que fue clickeado
  • _blank Abre el documento en una nueva ventana o pestaña
  • _parentAbre el documento en
