HTML Basico

Documentos HTML

Todos los documentos inician con una declaracion <!DOCTYPE html>

El documento HTML en si empieza con <html> </html>

La parte visible del documento es entre <body> </body>

Ejemplo

<!DOCTYPE html>
<html>
<body>

<h1>Mi primer encabezado</h1>
<p>Mi primer parrafo</p>
</body>
</html>

La declaracion <!DOCTYPE>

La declaracion <!DOCTYPE> representa el tipo de documento y ayuda al buscador a mostrar la pagina web correctamente

Esta solo debe de aparecer una vez en toda la pagina, que es en el inicio de esta y antes de cualquier otra etiqueta HTML

La etiqueta no es sensible a las mayusculas

La declaracion <!DOCTYPE> para HTML5 es:

<!DOCTYPE html>

Encabezados HTML

Los encabezados en HTML estan definidos por <h1> a <h6>

<h1> representa el encabezado más importante mientras que <h6> representa el menos importante

Ejemplo

<h1>Este es un encabezado 1</h1>
<h2>Este es un encabezado 2</h2>
<h3>Este es un encabezado 3</h3>

Parrafos HTML

Los parrafos son definidos con una etiqueta <p>:

Ejemplo

<p>Este es un parrafo</p>
<p>Este es otro parrafo</p>

Links HTML

Los links son definidos por la etiqueta <a>:

Ejemplo

<a href="https://www.llensomlle.com">Este es un links</a>

El link de destino está especificado por el atributo href

Los atributos son usados para proveer informacion adicional a los elementos HTML

Imagenes HTML

Las imagenes son definidas por la etiqueta <img>:

El archivo fuente: <src>

Texto alternativo: <alt>, la w3c nos dice que es obligatorio poner este atributo, ya que es un texto que se le colocara a la imagen si por alguna razon no puede cargar y para la accesibilidad

Tamaño: <width> y <height>
