Atributos HTML

  • Todos los elementos pueden tener atributos
  • Los atributos proveen de informacion adicional sobre los elementos
  • Los atributos siempre estan especificados en la etiqueta de inicio
  • Ellos usualmente vienen en nombre y valos parecidos así: nombre="valor"

Atributo href

La etiqueta <a> define un hipervinculo.

El atributo hrefespecifica la URL del enlace

Atributo src

La etiqueta <img> se usa para insertar una imagen en una pagina HTML

EL atributo <src> especifica la ruta de la imagen que se mostrará

Hay tres formas para especificar la URL del atributo src

1- URL's Absolutas Links de una imagen externa que esta hosteada en otro sitio. Por ejemplo: src="https://llensomlle.com/imagen-test.png"

2- URL's Relativas Links de una imagen que esta hosteada en nuestro host. En esta, la URL no necesita el dominio. Si la URL inicia sin un diagonal (/) sera relativa a la pagina actual. Por ejemplo: src="img_dog.jpg". Si la URL inicia con una diagonal (/) esta sera relativa al dominio

Nota: Siempre es mejor usar las URL relativas, de esa forma no se rompera si llegamos a cambiar el dominio

Los atributos width y height

La etiqueta <img> tambien contiene los atributos width y height que especifican lo alto y ancho de la imagen (en pixeles)

El atributo alt

El atributo requerido alt para la etiqueta <img> especofoca un texto alterno para la imagen, si la imagen por alguna razon no puede ser mostrada. Esto puede ser por la conexion lenta o algun error en el atributo src o tambien para la accesibilidad en un lector de pantalla

El atributo style

El atributo style es usado para añadir estilos a un elemento, como el color, fuente, tamaño y más

El atributo lang

Tu siempre puedes incluir el atributo lang dentro de la etiqueta html para declarar el idioma de la página web. Esto esta destinado para ayudar a los motores de busqueda y navegadores

Los codigos de pais pueden ser añadidos en el atributo lang. En donde los primeros dos caracteres definen el lenguaje de la pagina y los ultimos dos definen el pais

Puedes ver todos los codigos de lenguaje en Referencia de codigo de lenguaje HTML de la W3C

El atributo title

El atributo title define informacion extra sobre un elemento

El valor del titulo se mostrara como un tooltip cuando coloques tu cursor sobre el elemento
