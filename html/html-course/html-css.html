CSS ahora mucho trabajo. Así puedes controlar el diseño de muchas paginas web a la vez

¿Que es CSS?

Cascading Style Sheet (CSS) es usado para dar formato o estilo a la pagina web

Con CSS tú puedes controlar el color, fuente, tamaño del textom espacio entre elementos, como se colocan y distribuyen los elementos, que imagen o color de fondo son usados, diferentes display para diferentes dispositivos y tamaños de pantalla, ademas de mucho mas

Usando CSS

CSS se puede añadir al documento de tres formas

  • Inline - usandoo el atributo style dentro de los elementos HTML
  • Internal - usando el elemento <style< en la seccion <head>
  • External - usando el elemento <link> para vincular un archivo CSS externo

Inline CSS

Inline CSS es usado para aplicar un estilo unico a un elemento HTML

Este usa el atributo style en un elemento HTML

Los siguientes ejemplos ajustan el color del elemento <h1> en azul y el texto del elemento <p> en rojo

Ejemplo

<h1 style="color:blue;">Un titulo azul</h1>
<p style="color:red;">Un parrafo rojo</p>

Internal CSS

Un CSS interno es definido para darle estilo a una sola pagina HTML

Este es definido en la seccion <head> con un elemento <style>

Los siguientes ejemplos ajustan el color de TODOS los <h1> (En la pagina) de color azul, y el color de TODOS los elementos <p> en color rojo. Adicionalmente la pagina se mostrara con un color "powderblue" de fondo

Ejemplo

<!DOCTYPE html>
<html>
<head>
<style>
<body {backgroud-color: powderblue;}>
<h1 {color: blue;}>
<p {color: red;}>
</style>
</head>
<body>

<h1>Este es un encabezado</h1>
<p>Este es un parrafo</p>

</body> </html>

External CSS

Un CSS externo es usado para definir el estilos de muchas paginas HTML

Para usar un CSS externo debemos añadir un link en la seccion <head> de la pagina HTML

Ejemplo

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="style.css">
</head>
<body>

<h1>Este es un encabezado</h1>
<p>Este es un parrafo</p>

</body> </html>

El CSS externo se puede escribir en cualquier editor de texto. El archivo no debe contener nada de codigo HTML ademas de que debe ser guardado con una extension .css

Asi es como el archivo "style.css" debe lucir

style.css

body {
backgroud-color: powderblue;
}
h1 {
color: blue;
}
p {
color: red;
}

Tip: Con un CSS externo, tu puedes cambiar los estilos de todo un sitio web, con solo cambiar un archivo

Colores, Fuentes y Tamaños

Aqui demostraremos algunas propiedades CSS de uso común

La propiedad CSS color define el color del texto

La propiedad font-family define la fuente

La propiedad font-size define el tamaño del texto

<!DOCTYPE html>
<html>
<head>
<style>
h1 {
  color: blue;
2   font-family: verdana;
  font-size: 300%;
}
p {
  color: red;
 font-family: courier;
  font-size: 160%;
}
</style>
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

Borde CSS

La propiedad CSS border define un borde alrededor de un elemento HTML

Nota: Puedes definir el borde de casi todos los elementos HTML

p {
border: 2px solid powderblue;
}

CSS Padding

La propiedad CSS padding define un espacio entre el texto y el borde

Ejemplo

p {
border: 2px solid powderblue;
padding: 30px;
}

CSS Margin

Esta propiedad define un margen afuera del borde

Ejemplo

p {
border: 2pc solid powderblue;
margin: 50px;
}

Link a un CSS externo

Se puede ahacer referencia a una hoja de estilo externa con una URL completa o una ruta relativa a nuestra pagina actual

Ejemplo

<link rel="stylesheet" href="https://www.llensomlle.com/css/styles.css">
