Parrafos HTML

Un parrafo siempre inica en una nueva linea y suele ser un bloque de texto

El elemento <p> define un parrafo

Un parrafo siempre inicia en una nueva linea, y los naegadores automaticamente añaden un espacio (Margin) antes y despues del parrafo

HTML Display

Tú no puedes estar seguro de como se mostrara HTML

Grandes o pequeñas pantallas, y ventanas redimensionadas pueden crear diferentes resultados

Con HTML tu no puedes cambiar la visualizacion agregando espacion o lineas adicionales en tu codigo HTML

Los navegadores automaticamente removeran cualquier espacio extra y lineas cuando la pagina sea mostrara

Reglas horizontales HTML

La etiqueta <hr> define un salto tematico en una pagina HTML y por lo general se muestra como una regla horizontal

Este elemento es usado para separar contenido (o definir un cambio) en una pagina HTML

La etiqueta <hr> es una etiqueta vacia por lo que no tiene etiqueta de cierre

El problema de los poemas

Cuando queremos colocar un poema o cancion al intenar formatear el texto de esa forma en nuestro documento HTML, en la pagina nos lo mostrara en una linea

Pero hay una solucion, la etiqueta <pre> que define el texto preformateado

El texto que este dentro de la etiqueta sera mostrado con una fuente de ancho fijo (Normalmente Courier) y conserva tanto espacion como saltos de linea
