Introduccion a CSS

¿Que es CSS?

  • CSS son las siglas de Cascading Style Sheet
  • CSS describe como los elementos HTML van a ser mostrados en la pantalla, el papel u otro medio
  • CSS ahorra mucho trabajo, Con eso puedes controlar el diseño de multiples paginas web
  • Hojas de estilo externas son almacenadas en archivos CSS

¿Por que usar CSS?

CSS es usado para definir el estilo para tu pagina web, diseño y variantespara diferentes dispositivos y tamalos de pantalla

CSS soluciona un gran problema

HTML NUNCA tuvo la intencion de contener etiquetas para dar formato a una pagina web

HTML fue creado para describir el contenido de una pagina web

Etiquetas como <font> y atributos de colores fueron añadidos en la especificacion HTML 3.1, al principio era una pesadilla para los desarrolladores. Desarrollar grandes sitios web donde las fuentes e informacion del color fueran añadidas una por una a cada pagina, se convirtio en un largo y costoso proceso

Para resolver este problema la W3C creo CSS

CSS elimino el formato de estilos de HTML
